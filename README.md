## 微笔记

微笔记是一款 iOS 平台的记录类 App 。主要功能包括用户日常生活中需要记录的内容，记账、便签等功能；

项目特点：

- 支持 MarkDown 语法
- 本地沙盒存储（sqlite3 ），不依赖网络

- 可进行云同步（Bomb后端云）
